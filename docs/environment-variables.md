# Environment Variables

please open .env file in root directory, and at least complete the following variables

```
HUMANCODE_APP_ID=
HUMANCODE_APP_KEY=
HUMANCODE_API_HOST=https://humancodeai.com
HUMANCODE_CALLBACK_URL=

# telegram
TELEGRAM_BOT_NAME=
TELEGRAM_BOT_TOKEN=
TELEGRAM_WEBAPP_LINK=

# ton
TON_CENTER_API_KEY=
```
